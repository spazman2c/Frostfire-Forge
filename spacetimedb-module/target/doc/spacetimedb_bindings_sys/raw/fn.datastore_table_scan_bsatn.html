pub unsafe extern "C" fn datastore_table_scan_bsatn(
    table_id: TableId,
    out: *mut RowIter,
) -> u16
Expand description

Starts iteration on each row, as BSATN-encoded, of a table identified by table_id.

On success, the iterator handle is written to the out pointer. This handle can be advanced by row_iter_bsatn_advance.

This function does not trap.

  • NO_SUCH_TABLE, when table_id is not a known ID of a table.
