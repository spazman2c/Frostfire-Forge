Add a reducer to the in-progress module. Accepts a ProductType of reducer arguments for convenience. The ProductType need not be registered in the typespace.

Importantly, if the reducerโs first argument is a ReducerContext, that information should not be provided to this method. As far as the module definition is concerned, the reducerโs arguments start with the first non-ReducerContext argument.
