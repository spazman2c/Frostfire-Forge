}
Expand description

A data structure representing the definition of a database table.

This struct holds information about the table, including its name, columns, indexes, constraints, sequences, type, and access rights.

Fields§

§table_name: Box<str>

The name of the table.

§columns: Vec<RawColumnDefV8>

The columns of the table. The ordering of the columns is significant. Columns are frequently identified by ColId, that is, position in this list.

§indexes: Vec<RawIndexDefV8>

The indexes on the table.

§constraints: Vec<RawConstraintDefV8>

The constraints on the table.

§sequences: Vec<RawSequenceDefV8>

The sequences attached to the table.

§table_type: StTableType

Whether the table was created by a user or by the system.

§table_access: StAccess

The visibility of the table.

§scheduled: Option<Box<str>>

If this is a schedule table, the reducer it is scheduled for.
